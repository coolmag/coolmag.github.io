# Сайт-визитка для Telegram-канала "Aurora Vibe Free Music"

Этот репозиторий содержит простой одностраничный сайт (`index.html`) для продвижения вашего Telegram-канала.

Вы можете открыть файл `index.html` в любом браузере, чтобы увидеть, как он выглядит локально.

---

## Как бесплатно опубликовать сайт в интернете с помощью GitHub Pages

GitHub Pages — это бесплатный сервис от GitHub, который позволяет размещать простые сайты напрямую из вашего репозитория. Следуйте этим шагам, чтобы опубликовать вашу страницу.

### Шаг 1: Создайте аккаунт на GitHub

Если у вас еще нет аккаунта, зарегистрируйтесь на [github.com](https://github.com).

### Шаг 2: Создайте специальный репозиторий

1.  Войдите в свой аккаунт. В правом верхнем углу нажмите на `+` и выберите **New repository** (Новый репозиторий).

2.  Это самый важный шаг. Назовите репозиторий в формате **`ваше_имя_пользователя.github.io`**.
    *   Обязательно замените `ваше_имя_пользователя` на ваш реальный никнейм на GitHub. Например, если ваш ник `octocat`, репозиторий должен называться `octocat.github.io`.

3.  Убедитесь, что репозиторий **публичный (Public)**.

4.  Нажмите кнопку **Create repository** (Создать репозиторий).

### Шаг 3: Загрузите ваш сайт

1.  Перейдите в созданный репозиторий.

2.  Нажмите на кнопку **Add file** (Добавить файл) и выберите **Upload files** (Загрузить файлы).

3.  Перетащите ваш файл `index.html` в область загрузки или выберите его на компьютере.

4.  Нажмите кнопку **Commit changes** (Сохранить изменения).

### Шаг 4: Готово!

Подождите несколько минут (иногда до 10). Ваш сайт будет автоматически опубликован и доступен в интернете по адресу:

**`https://ваше_имя_пользователя.github.io`**

Теперь вы можете делиться этой ссылкой с кем угодно!
