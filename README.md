## 🎵 О проекте

Этот проект представляет собой одностраничный сайт (landing page), созданный с целью:

- **Привлечь внимание** к Telegram-каналу с помощью чистого и анимированного дизайна.
- **Предоставить прямую ссылку** для быстрой подписки на канал: [t.me/coolmagDJ](https://t.me/coolmagDJ).
- **Обеспечить базовое присутствие** в поисковых системах для органического трафика.

## 🛠️ Стек технологий

Сайт создан с использованием базовых веб-технологий, что обеспечивает максимальную скорость загрузки и простоту.

- **HTML5**
- **CSS3** (включая Flexbox, Animations)
- **Google Fonts** (шрифт Montserrat)

## 🌐 Развертывание и Публикация

<details>
<summary><strong>Как бесплатно опубликовать сайт (Инструкция)</strong></summary>

Сайт можно легко и бесплатно разместить в интернете с помощью **GitHub Pages**.

### Шаг 1: Создайте репозиторий
- Войдите на GitHub.
- Создайте **новый публичный (Public)** репозиторий.
- **ВАЖНО:** Имя репозитория должно в точности совпадать с вашим ником в формате: `ВАШ_НИК.github.io`.
  - *(Например, если ваш ник — `coolmag`, имя репозитория должно быть `coolmag.github.io`)*.

### Шаг 2: Загрузите файл
- В созданном репозитории нажмите **Add file** -> **Upload files**.
- Перетащите в окно браузера файл `index.html`.
- Нажмите **Commit changes**.

### Шаг 3: Готово!
- Подождите 5-10 минут.
- Ваш сайт станет доступен по адресу: **`https://ВАШ_НИК.github.io`**

</details>
