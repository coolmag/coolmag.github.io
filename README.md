# 🚀 Aurora Vibe Free Music - Промо-сайт

Этот репозиторий содержит код для простого, но стильного сайта-визитки, созданного для продвижения Telegram-канала **"Aurora Vibe Free Music"**.

> Вы можете посмотреть, как сайт выглядит локально, открыв файл `index.html` в любом браузере.

---

## ✅ Как опубликовать сайт бесплатно (Инструкция)

Сайт можно легко и бесплатно разместить в интернете с помощью **GitHub Pages**.

### Шаг 1: Создайте репозиторий
- Войдите на GitHub.
- Создайте **новый публичный (Public)** репозиторий.
- **ВАЖНО:** Имя репозитория должно в точности совпадать с вашим ником в формате: `ВАШ_НИК.github.io`.
  - *(Например, если ваш ник на GitHub — `coolmag`, имя репозитория должно быть `coolmag.github.io`)*.

### Шаг 2: Загрузите файлы
- В созданном репозитории нажмите **Add file** -> **Upload files**.
- Перетащите в окно браузера файл `index.html` из этого проекта.
- Нажмите **Commit changes**.

### Шаг 3: Готово!
- Подождите 5-10 минут.
- Ваш сайт станет доступен по адресу: **`https://ВАШ_НИК.github.io`**

---

###  Troubleshooting: Ошибка 404 (Сайт не найден)

Если вы видите ошибку 404, проверьте эти пункты:

1.  **Имя репозитория:** Убедитесь, что имя на 100% совпадает с вашим ником (`ник.github.io`). Это самая частая причина ошибки.
2.  **Главный файл:** В корне репозитория должен лежать файл с именем `index.html` (все буквы строчные).
3.  **URL адрес:** Проверьте, что вы открываете правильный адрес (`https://ник.github.io`), а не ссылку на сам репозиторий.
