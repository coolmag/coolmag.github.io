# Aurora Vibe - Web Player & Landing Page
🎧 AI Music Station & Web Player


Твой персональный музыкальный хаб: веб-плеер + Telegram AI-DJ.
Слушай, управляй, взаимодействуй. Полностью бесплатно и на нейросетях.
🚀 О проекте
Это не просто плеер. Это экосистема, объединяющая веб-интерфейс и Telegram-канал, где за контент отвечает AI DJ.
Что внутри:
Web Player: Чистый HTML/CSS/JS интерфейс для стриминга.
AI DJ (Telegram): Автоматическая генерация и подборка треков, посты в реальном времени.
Интерактив: Голосования за треки, заказы музыки через бота.
Open Source: Весь стек на Python, легко кастомизировать.
🛠 Технологии
Backend: Python (Flask/FastAPI)
Frontend: HTML5, CSS3 (Modern UI), JavaScript
Automation: Telegram Bot API, AI-models для генерации подборок
Deployment: Railway.app (CI/CD через GitHub)
📦 Быстрый старт
Клонируй репозиторий:
bash
git clone https://github.com
cd название-проекта
Используйте код с осторожностью.

Настрой окружение:
Создай файл .env и добавь ключи:
env
TELEGRAM_TOKEN=твой_токен
DB_URL=твоя_бд
Используйте код с осторожностью.

Запусти локально:
bash
pip install -r requirements.txt
python main.py
Используйте код с осторожностью.

🌍 Развертывание
Проект полностью готов к деплою на Railway:
Подключи GitHub репозиторий к Railway.
Добавь переменные окружения в панели управления.
Система сама соберет и запустит проект (Procfile/Dockerfile уже на борту).
