## 🧬 Особенности экосистемы

> **Aurora Vibe** — это не просто плеер. Это живой организм, где нейросети создают контент, а вы им управляете.

*   **⚡ Web Interface:** Ультра-легкий плеер на чистом JS/CSS с поддержкой медиа-сессий.
*   **🤖 AI DJ (Neural Engine):** Автоматизированная генерация и подборка треков без участия человека.
*   **📡 Real-time Sync:** Мгновенная публикация постов в TG-канал через веб-панель.
*   **🗳️ Community Driven:** Голосования и заказы музыки в реальном времени через бота.
