**Aurora Vibe** — это не просто радио. Это живая экосистема, где нейросетевой движок генерирует и подбирает музыку, а вы управляете эфиром прямо из Telegram. Легкий веб-интерфейс и мощный AI-бот работают в идеальной синхронизации.

## 🧬 Ключевые возможности

| Функция | Описание |
| :--- | :--- |
| 🤖 **AI DJ Engine** | Полностью автономный подбор треков, генерация постов и ведение эфира 24/7. |
| ⚡ **Web Interface** | Минималистичный плеер на чистом JS/CSS с фоновым видео и поддержкой медиа-сессий. |
| 📡 **Real-time Sync** | Веб-плеер и Telegram-канал всегда показывают один и тот же трек. |
| 🎵 **Community Driven** | Интерактивное взаимодействие: заказ треков и участие в опросах через бота. |

## 🛠️ Технологии

- **Backend**: `Python 3.11+` (Asyncio, AIOgram, FastAPI)
- **Frontend**: `HTML5` • `CSS3` (Glassmorphism, Neon UI) • `Vanilla JS`
- **Инфраструктура**: `Railway` (хостинг бэкенда) • `GitHub Pages` (хостинг фронтенда)
- **Автоматизация**: `Telegram Bot API` • `Generative AI Models`

## 🚀 Быстрый старт

1.  **Клонировать репозиторий:**
    ```bash
    git clone https://github.com/YourUsername/AuroraVibe.git
    cd AuroraVibe
    ```
2.  **Настроить окружение:**
    - Создайте файл `.env` на основе `.env.example`.
    - Установите зависимости: `pip install -r requirements.txt`.
3.  **Запустить:**
    - Для локальной разработки запустите основной Python-скрипт.
