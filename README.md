# Aurora Vibe - Web Player & Landing Page

[![Статус проекта: активен](https://img.shields.io/badge/статус-активен-success.svg)](https://coolmag.github.io/)

Современный веб-интерфейс для Telegram-бота "Aurora Vibe", сочетающий в себе функциональный плеер и стильную целевую страницу с видео-фоном.

![Скриншот проекта](https://github.com/coolmag/coolmag.github.io/assets/136006471/080c4103-68f7-43c2-a083-d9d300465228)

## 🚀 О проекте

Этот проект служит визуальной оболочкой и основным пользовательским интерфейсом для нейросетевой радиостанции "Aurora Vibe". Он превращает простую ссылку на бота в полноценное веб-приложение, где пользователи могут напрямую взаимодействовать с плеером на фоне динамического видео.

## ✨ Ключевые особенности

- **Живой фон:** Полноэкранное фоновое видео для создания атмосферы.
- **Встроенный веб-плеер:** Прямая интеграция функционального плеера с Railway через `iframe`.
- **"Стеклянный" интерфейс (Glassmorphism):** Полупрозрачные размытые блоки создают ощущение глубины и современный вид.
- **Адаптивный дизайн:** Корректно отображается на десктопных и мобильных устройствах.
- **SEO-оптимизация:** Прописаны базовые мета-теги и Open Graph теги для корректного отображения в поисковиках и социальных сетях.

## 🛠️ Стек технологий

- **Лендинг:** HTML5, CSS3 (CSS Variables, Flexbox, Media Queries)
- **Плеер (бэкенд):** Python (FastAPI), развернут на **Railway**.
- **Хостинг лендинга:** **GitHub Pages**.

## ⚙️ Установка и развертывание

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/coolmag/coolmag.github.io.git
   ```
2. **Подготовьте файлы:**
   - Убедитесь, что в корневой папке находятся файлы `index.html` и `avatar.mp4`.
3. **Разверните на GitHub Pages:**
   - Загрузите содержимое в репозиторий с названием `ВАШ_НИК.github.io`.
   - Сайт будет доступен по адресу `https://ВАШ_НИК.github.io`.

**Важно:** Для корректной работы эффекта прозрачности, в коде самого плеера (на Railway) фон должен быть установлен как `transparent`.

## 🔮 Возможные улучшения (Roadmap)

- **Аудиореактивный фон:** Интеграция Web Audio API для создания фона, пульсирующего в такт музыке.
- **Голосовое управление:** Добавление интерфейса для управления плеером с помощью голоса.
- **Интеграция с Telegram Mini Apps:** Адаптация для запуска в виде нативного приложения внутри Telegram.
